# ToDo-API-BSBO (FastAPI) — Практика 1

Учебное API на FastAPI для списка задач с классификацией по матрице Эйзенхауэра (Q1–Q4).

Технологии: Python, FastAPI, Uvicorn.

## Запуск
1) Установить зависимости:
```bash
pip install -r requirements.txt
```
Запустить сервер:
```bash
uvicorn main:app --reload
```

Swagger UI:
http://127.0.0.1:8000/docs
